# IBus 插件结构说明

## 插件目录结构

```
my-input-engine/
├── src/
│   ├── my-input-engine.c          # 引擎实现
│   ├── my-input-engine.h          # 引擎头文件
│   ├── my-input-engine-desc.c     # 引擎描述实现
│   ├── my-input-engine-desc.h     # 引擎描述头文件
│   └── Makefile.am                # Makefile 规范
├── docs/
│   ├── my-input-engine.xml        # 引擎配置文件
│   └── README.md                  # 引擎说明文档
├── data/
│   ├── icons/
│   │   └── my-input-engine.png    # 引擎图标
│   └── schemas/
│       └── org.freedesktop.ibus.engine.my-input-engine.gschema.xml
├── tests/
│   └── test-my-input-engine.c     # 测试代码
└── Makefile                       # 构建文件
```

## 核心文件说明

### 1. my-input-engine.c

输入法引擎的主要实现文件。

#### 结构定义

```c
typedef struct _MyInputEngine MyInputEngine;
typedef struct _MyInputEngineClass MyInputEngineClass;
typedef struct _MyInputEnginePrivate MyInputEnginePrivate;

struct _MyInputEngine {
    IBusEngineSimple parent;
};

struct _MyInputEnginePrivate {
    // 引擎私有数据
    guint32 *key_history;
    gint key_history_len;
    gchar *preedit_text;
    guint preedit_len;
    gboolean preedit_visible;
    // ...
};
```

#### 必需函数

```c
// 类型定义
GType my_input_engine_get_type (void);

// 引擎初始化
void my_input_engine_init (MyInputEngine *engine);

// 类初始化
void my_input_engine_class_init (MyInputEngineClass *klass);

// 销毁函数
static void my_input_engine_destroy (IBusEngineSimple *simple);
```

#### 回调函数（必需）

```c
// 按键处理

// 候选词点击

    (IBusEngine *engine,
     guint index,
     guint button,
     guint state);

// 上下文管理
static void my_input_engine_set_surrounding_text
    (IBusEngine *engine,
     IBusText *text,
     guint cursor_index,
     guint anchor_pos);

static void my_input_engine_get_surrounding_text
    (IBusEngine *engine,
     IBusText **text,

// 文本更新
```

### 2. my-input-engine.h

引擎头文件，包含类型定义和函数声明。

```c
#ifndef __MY_INPUT_ENGINE_H__
#define __MY_INPUT_ENGINE_H__

#include "ibus.h"
#include "ibusenginesimple.h"

G_BEGIN_DECLS

#define MY_TYPE_INPUT_ENGINE             \
    (my_input_engine_get_type ())
#define MY_INPUT_ENGINE(obj)             \
    (MY_TYPE_INPUT_ENGINE (obj))
#define MY_IS_INPUT_ENGINE(obj)          \
    (MY_TYPE_INPUT_ENGINE (obj))

typedef struct _MyInputEngine MyInputEngine;
typedef struct _MyInputEngineClass MyInputEngineClass;
typedef struct _MyInputEnginePrivate MyInputEnginePrivate;

GType          my_input_engine_get_type       (void);

IBusEngine    *my_input_engine_new            (const gchar *engine_name,
                                                const gchar *object_path,
                                                GDBusConnection *connection);

G_END_DECLS

#endif
```

### 4. my-input-engine.xml

引擎配置文件，定义引擎元数据。

```xml
<?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE engine>
<engine>
    <!-- 基本信息 -->
    <name>my-input-engine</name>
    <description>Engine Description</description>
    <author>Author Name</author>
    <license>LGPL</license>
    <version>1.0.0</version>
    
    <!-- 支持的语言 -->
    <lang>en</lang>
    <lang>zh_CN</lang>
    <lang>zh_TW</lang>
    
    <!-- 引擎能力 -->
    <capability>Focus</capability>
    <capability>PreeditText</capability>
    <capability>LookupTable</capability>
    
    <!-- 属性配置 -->
    <property>
        <name>InputMode</name>
        <default>Normal</default>
    </property>
</engine>
```

## Makefile.am 规范

```makefile
# 源文件列表
noinst_LTLIBRARIES = libmy-input-engine.la

libmy_input_engine_la_SOURCES = \
    my-input-engine.c \
    my-input-engine-desc.c

libmy_input_engine_la_LIBTOOLFLAGS = --tag=shared
libmy_input_engine_la_LDFLAGS = -version-info 1:0:0

# 头文件
noinst_HEADERS = \
    my-input-engine.h \
    my-input-engine-desc.h

# 配置文件
enginesdir = $(libdir)/ibus/engines
engines_DATA = my-input-engine.xml

# 数据文件
datadir = $(datadir)/ibus/engines
data_DATA = my-input-engine.xml
```

## 构建步骤

### 1. 准备构建环境

```bash
autoreconf -i
```

### 2. 配置

```bash
./configure --prefix=/usr
```

### 3. 编译

```bash
make
```

### 4. 安装

```bash
make install
```

## 常见问题

### Q1: 引擎无法加载

A: 检查配置文件路径和语法：
```bash
ls /usr/share/ibus/engines/my-input-engine.xml
ibus-enumerate-engines
```

### Q2: 按键事件未处理

A: 检查回调函数是否已注册：
```c
engine_class->process_key_event = my_input_engine_process_key_event;
```

### Q3: 候选词未显示

A: 检查查找表更新：
```c
IBusLookupTable *table = ibus_lookup_table_new (engine);
ibus_engine_update_lookup_table (engine, table, TRUE);
```
