# IBus API 参考

## 概述

本文档详细说明了 IBus 框架的核心 API 接口，包括输入引擎开发所需的函数、信号和回调。

## 核心接口

### IBusEngine

输入方法引擎抽象类，提供输入法引擎的基础功能。

#### 回调函数

| 函数 | 参数 | 返回值 | 说明 |
|------|------|--------|------|
| `process_key_event` | IBusEngine *engine, guint keyval, guint keycode, guint state | gboolean | 处理键盘按键事件 |
| `focus_in` | IBusEngine *engine | void | 输入法获得焦点 |
| `focus_out` | IBusEngine *engine | void | 输入法失去焦点 |
| `reset` | IBusEngine *engine | void | 重置输入法状态 |
| `enable` | IBusEngine *engine | void | 启用输入法 |
| `disable` | IBusEngine *engine | void | 禁用输入法 |
| `set_cursor_location` | IBusEngine *engine, gint x, gint y, gint w, gint h | void | 设置光标位置 |
| `set_capabilities` | IBusEngine *engine, guint caps | void | 设置客户端能力 |
| `page_up` | IBusEngine *engine | void | 向上翻页 |
| `page_down` | IBusEngine *engine | void | 向下翻页 |
| `cursor_up` | IBusEngine *engine | void | 光标上移 |
| `cursor_down` | IBusEngine *engine | void | 光标下移 |
| `candidate_clicked` | IBusEngine *engine, guint index, guint button, guint state | void | 候选词点击 |
| `set_surrounding_text` | IBusEngine *engine, IBusText *text, guint cursor_index, guint anchor_pos | void | 设置上下文文本 |
| `get_surrounding_text` | IBusEngine *engine, IBusText **text, guint *cursor_pos, guint *anchor_pos | void | 获取上下文文本 |
| `delete_surrounding_text` | IBusEngine *engine, gint offset, guint nchars | void | 删除上下文文本 |
| `set_content_type` | IBusEngine *engine, guint purpose, guint hints | void | 设置内容类型 |
| `process_hand_writing_event` | IBusEngine *engine, const gdouble *coordinates, guint coordinates_len | void | 处理手写事件 |
| `cancel_hand_writing` | IBusEngine *engine, guint n_strokes | void | 取消手写 |

#### 主要函数

| 函数 | 参数 | 返回值 | 说明 |
|------|------|--------|------|

| `ibus_engine_new_with_type` | engine_type, engine_name, object_path, connection | IBusEngine * | 使用类型创建引擎 |
| `ibus_engine_commit_text` | engine, text | void | 提交文本到客户端 |

| `ibus_engine_update_preedit_text_with_mode` | engine, text, cursor_pos, visible, mode | void | 更新预编辑文本（带模式） |
| `ibus_engine_show_preedit_text` | engine | void | 显示预编辑区 |
| `ibus_engine_hide_preedit_text` | engine | void | 隐藏预编辑区 |
| `ibus_engine_update_auxiliary_text` | engine, text, visible | void | 更新辅助文本 |
| `ibus_engine_update_lookup_table` | engine, lookup_table, visible | void | 更新候选词表 |
| `ibus_engine_update_lookup_table_fast` | engine, lookup_table, visible | void | 快速更新候选词表 |
| `ibus_engine_show_lookup_table` | engine | void | 显示候选词表 |
| `ibus_engine_hide_lookup_table` | engine | void | 隐藏候选词表 |
| `ibus_engine_forward_key_event` | engine, keyval, keycode, state | void | 转发按键事件 |
| `ibus_engine_register_properties` | engine, prop_list | void | 注册属性 |
| `ibus_engine_update_property` | engine, prop | void | 更新属性 |
| `ibus_engine_get_surrounding_text` | engine, text, cursor_pos, anchor_pos | void | 获取上下文文本 |
| `ibus_engine_get_content_type` | engine, purpose, hints | void | 获取内容类型 |

## IBus 能力标志

| 标志 | 值 | 说明 |
|------|-----|------|
| `IBUS_CAP_FOCUS` | 0x0001 | 支持焦点 |
| `IBUS_CAP_PREEDIT_TEXT` | 0x0002 | 支持预编辑文本 |
| `IBUS_CAP_AUXILIARY_TEXT` | 0x0004 | 支持辅助文本 |
| `IBUS_CAP_LOOKUP_TABLE` | 0x0008 | 支持查找表 |
| `IBUS_CAP_PROPERTY` | 0x0010 | 支持属性 |

## 消息类型

| 标志 | 值 | 说明 |

## 示例代码

### 基本的按键处理

```c
static gboolean
my_engine_process_key_event (IBusEngine *engine,
                             guint keyval,
                             guint keycode,
                             guint state)
{
    MyInputEngine *my_engine = MY_INPUT_ENGINE (engine);
    MyInputEnginePrivate *priv = MY_INPUT_ENGINE_GET_PRIVATE (my_engine);
    IBusText *text;
    guint cursor_pos;
    
    // 处理按键逻辑
    // ...
    
    // 更新预编辑文本
    text = ibus_text_new_from_string (priv->preedit_text);
    cursor_pos = priv->cursor_pos;
    ibus_engine_update_preedit_text (engine, text, cursor_pos, TRUE);
    g_object_unref (text);
    
    return TRUE;
}
```

### 提交候选词

```c
static void
my_engine_candidate_clicked (IBusEngine *engine,
                             guint index,
                             guint button,
                             guint state)
{
    MyInputEngine *my_engine = MY_INPUT_ENGINE (engine);
    MyInputEnginePrivate *priv = MY_INPUT_ENGINE_GET_PRIVATE (my_engine);
    IBusText *text;
    
    // 获取候选词
    if (index < priv->candidates->len) {
        // 提交候选词
        text = ibus_text_new_from_string (priv->candidates[index]);
        ibus_engine_commit_text (engine, text);
        g_object_unref (text);
    }
}
```
